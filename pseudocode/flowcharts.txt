FLOWCHART 5: Six-Frame ORF Scanner
────────────────────────────────────────────────────────────────────────

    [Start: receive sequence, start codons, minimum length]
       |
       v
    [Compute reverse complement of sequence]
       |
       v
    [For each frame in [0, 1, 2]:]
       |
       +---> [Scan forward strand at this frame offset]
       |
       +---> [Scan reverse complement at this frame offset]
       |
       v
    [Split sequence into codons at frame offset]
       |
       v
    [Any codons?]
       /      \
     Yes        No
      |          |
      |          v
      |    [Skip this frame]
      |
      v
    [Find all positions matching a start codon]
       |
       v
    [For each start codon position (earliest first):]
       |
       v
    [Find next in-frame stop codon]
       |
       v
    [Stop codon found?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — ORF is incomplete]
      |
      v
    [Calculate ORF length in nucleotides]
       |
       v
    [Length >= minimum length?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Skip — ORF too short]
      |
      v
    [Has a longer ORF already been recorded for this stop position?]
       /          \
     Yes           No
      |             |
      v             v
    [Skip —    [Convert coordinates to forward-strand positions]
     shorter]  [Record ORF: strand, frame, start, end,
                length, start codon]
       |             |
       +------+------+
              |
              v
    [Move to next start codon position]
              |
              v
    [All six frames scanned?]
       /          \
     Yes           No
      |             |
      |             v
      |       [Continue to next frame]
      |
      v
    [Return flat list of all ORF records]
